# Study Friend

![Study Friend UI](https://i.imgur.com/DJ28WAY.jpg)

## Description
A very effective studying method is creating questions based on the material you just learned, then answering those questions. 
That is what this application allows users to do — write questions and the answers to those questions. 
Built with Razor Pages with Entity Framework Core in ASP.NET Core. ([Intro to Razor Pages](https://docs.microsoft.com/en-us/aspnet/core/razor-pages/index?view=aspnetcore-3.0&tabs=visual-studio))

## Getting Started
These instructions rely heavily on Visual Studio, however there are options available.
Please reference the Razor Pages tutorials for more information: 
- [Visual Studio Instructions](https://docs.microsoft.com/en-us/aspnet/core/tutorials/razor-pages/razor-pages-start?view=aspnetcore-3.0&tabs=visual-studio)
- [VS Code Instructions](https://docs.microsoft.com/en-us/aspnet/core/tutorials/razor-pages/razor-pages-start?view=aspnetcore-3.0&tabs=visual-studio-code)

### Prerequisites
- [Visual Studio](https://visualstudio.microsoft.com/downloads/)

### Installing
- Clone the repository ([https://github.com/dacrands/StudyFriend.git]) and open the solution in Visual Studio. 
- Build the solution
- Run `Update-Database` to create the database using the last migration *(Note: There is currently no seed data at this moment)*

## Deployment
This web app is published to Azure Web Services. Please reference the following tutorial for instructions:
[Publish an ASP.NET Core app to Azure with Visual Studio](https://docs.microsoft.com/en-us/aspnet/core/tutorials/publish-to-azure-webapp-using-vs?view=aspnetcore-3.0)
